Photofolio (More than just a portfolio)

This project involved designing and developing an advanced platform tailored for photographers, focusing on seamless portfolio management, interactive client engagement, and specialized service offerings. As the lead developer, I architected the backend using Node.js and MongoDB, leveraging Mongoose for efficient data modeling. The project allowed me to navigate the full software development lifecycle while ensuring high standards of usability and performance

Self-Driving Vehicle

For this capstone project, I explored the application of computer vision and artificial intelligence to build an autonomous self-driving car system. Using Convolutional Neural Networks (CNNs), the system processed real-time video input to make driving decisions. The solution was engineered with Python, CUDA, and Arduino C, and trained using the CamVid dataset with the FastAI framework. The setup ran on a NVIDIA Jetson Nano chip and Linux environment, interfaced and programmed via Visual Studio Code. This project deepened my expertise in deep learning, embedded systems, and practical AI deployment

Air Quality Monitoring System (with Nokia)

Partnering with Nokia, I developed an IoT-based air quality monitoring solution employing the MQ-135 air sensor. The system continuously tracked and analyzed pollutant levels, with processed data visualized through comprehensive dashboards. Technologies included NodeMCU for real-time sensor communication, Embedded C for firmware, and Python for data visualization. This project underscored the impact of integrating sensor hardware with cloud-enabled analytics to drive adaptive responses in environmental monitoring

Smart Key System

The Smart Key System project focused on creating an IoT-enabled vehicle security solution. The device, installed in a vehicle, scanned smart keys and verified user credentials including insurance and driving license validity, by interfacing with a backend user database. Only upon successful validation would the vehicle unlock, adding a robust layer of safety and compliance. My role included designing the system architecture and developing the communication protocol between hardware and server

Threat Intelligence and Vulnerability Management (with Hewlett Packard Enterprise)

During this R&D-focused project in association with HPE, I developed "Threat Ripper," a command-line utility that aggregates vulnerability data from platforms such as VirusTotal and various threat intelligence APIs. This tool streamlined the vulnerability assessment workflow for security analysts, offering actionable insights for software system protection and cyber risk detection. The experience honed my programming skills while exposing me to the fundamentals of cybersecurity and vulnerability management in enterprise contexts
